## 🆘 如果页面打不开

### 检查步骤：
1. **确认应用运行状态**:
   ```bash
   netstat -an | findstr :5000
   ```

2. **重启应用**:
   ```bash
   cd SmartRecruit_LLM-main
   python run.py
   ```

3. **检查端口占用**:
   ```bash
   netstat -ano | findstr :5000
   ```

4. **访问演示页面**:
   - 直接访问: `http://localhost:5000/demo/job_search`

### 常见问题：
- **500错误**: 通常是数据库连接问题，演示页面会自动使用模拟数据
- **404错误**: 检查URL是否正确
- **连接被拒绝**: 确认应用是否正在运行
